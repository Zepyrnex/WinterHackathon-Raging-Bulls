# Voltify

## Description
Smart Energy Consumption Optimizer is a web-based application that helps households understand and reduce electricity usage.

It visualizes power consumption data, highlights usage patterns, and provides simple recommendations to minimize unnecessary energy consumption.

## Features

- Consumption history and trend analysis using graphs
- Energy-saving recommendations based on usage patterns
