# **App Name**: Voltify

## Core Features:

- Real-time kWh Monitoring: Display real-time power consumption data using a line chart (Chart.js), updating every 10 seconds, visualizing kilowatt-hours (kWh).
- Consumption Views: Offer daily, weekly, and monthly views of power consumption data via interactive tabs.
- Appliance Breakdown: Visualize appliance-specific power consumption using a pie chart (e.g., AC 45%, Lights 15%).
- Peak Usage Alerts: Provide real-time alerts when power consumption exceeds a defined threshold (e.g., >3kW), suggesting actions to reduce usage.
- Cost Calculation: Calculate and display the cost of electricity usage based on a user-defined rate (₹8/kWh), showing daily and monthly costs.
- Efficiency Grading: Grade the home's energy efficiency compared to average Indian homes, using an A-F scale. Display tips for improvement.
- Smart Suggestions: AI-powered tool to generate energy-saving recommendations (e.g., delay laundry by 2h to save ₹80/wk), learning user habits over time and using a tool to dynamically incorporate household data into suggestions .

## Style Guidelines:

- Background color: Dark navy blue (#1A237E), fostering a modern, serious feel. Heavily desaturated; analogous to the primary.
- Primary color: Vibrant blue (#2962FF) for the main elements and calls to action; high saturation for clear visual impact against the dark background.
- Accent color: Purple (#6200EA), analogous to blue, to highlight key interactive elements and provide contrast.
- Font pairing: 'Inter' (sans-serif) for body text, and 'Poppins' (sans-serif) for headers, providing a balance of readability and modern aesthetics.
- Use clean, minimalist vector icons to represent different appliances and data points.
- Implement a mobile-first, responsive layout with gradient-filled cards and glassmorphism effects for a visually appealing interface.
- Use smooth loading spinners and subtle transitions for a polished and engaging user experience.
